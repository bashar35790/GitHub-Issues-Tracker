## JavaScript Concepts

### 1️⃣ What is the difference between `var`, `let`, and `const`?

`var`, `let`, and `const` are used to declare variables in JavaScript, but they behave differently.

- **var**
  - Function scoped
  - Can be redeclared and updated
  - Older way of declaring variables

- **let**
  - Block scoped
  - Can be updated but cannot be redeclared in the same scope
  - Preferred for variables that will change

- **const**
  - Block scoped
  - Cannot be updated or redeclared
  - Used for values that should remain constant

### 2️⃣ What is the spread operator (...)?

Ans: The spread operator (...) is used to expand elements of an array or object. It is commonly used for copying, merging, or passing elements.

### 3️⃣ What is the difference between map(), filter(), and forEach()?

These are array methods used to iterate over arrays.

1.map()
Creates a new array
Transforms each element

2.filter()
Creates a new array
Returns elements that match a condition

3.forEach()
Loops through an array
Does not return a new array


### 4️⃣ What is an arrow function?

An arrow function is a shorter way to write functions in JavaScript. It was introduced in ES6 and uses the => syntax.

### 5️⃣ What are template literals?

Template literals are a way to create strings using backticks (`). They allow embedding variables and expressions inside strings using ${}.
